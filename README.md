# Sistema de Gestión Estudiantil

Prototipo frontend desarrollado como parte de la etapa de análisis y diseño del proyecto de Programacion FullStack y Electiva IA.

## Descripción

Este proyecto consiste en el diseño de una interfaz para un Sistema de Gestión Estudiantil cuyo objetivo es centralizar la administración de estudiantes, cursos, asistencia y calificaciones dentro de una única plataforma.

Actualmente se encuentra implementada la primera versión del frontend mediante HTML y CSS, simulando la experiencia de uso del sistema.

## Objetivo

Diseñar una interfaz moderna, organizada y responsive que permita visualizar cómo funcionará el sistema una vez implementado completamente.

## Funcionalidades Implementadas

### Autenticación

* Pantalla de inicio de sesión.
* Formulario de acceso.
* Recuperación de contraseña (visual).

### Dashboard

* Estadísticas generales.
* Indicadores académicos.
* Alertas de rendimiento e inasistencias.

### Gestión de Estudiantes

* Tabla de estudiantes.
* Estado de matrícula.
* Acciones de administración.

### Gestión de Cursos

* Visualización de cursos.
* Información de docentes.
* Cantidad de estudiantes inscritos.

### Control de Asistencia

* Registro de asistencia por estudiante.
* Selección de fecha y curso.

### Gestión de Calificaciones

* Registro de notas.
* Cálculo visual de promedio final.

### Perfil de Usuario

* Información personal.
* Actualización de datos.
* Configuración básica de cuenta.

## Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript Vanilla (interacciones básicas)

## Características del Diseño

* Interfaz tipo Institucional Dashboard.
* Sidebar de navegación.
* Componentes reutilizables.

* Experiencia de usuario enfocada en entornos educativos.
